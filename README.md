# From Stethoscope to Terminal: A Doctor's Daily Life with AI Agents

> Keynote presentation and live demo scripts

## Overview

A keynote talk demonstrating how a clinician-researcher integrates AI agents into daily medical practice — from patient care to academic research.

## Contents

- **`AI_Hub_Keynote_From_Stethoscope_to_Terminal.pptx`** — 30-slide keynote deck with full speaker notes
- **`AI_Hub_Keynote_Demo_Scripts.docx`** — step-by-step live demo scripts (Demo 1–3, Hands-On, Backup Plans, Pre-Talk Checklist)

## Talk Abstract

Modern medicine generates more data than any single physician can process. This keynote documents one doctor's journey from paper charts to an AI-augmented workflow: building personal AI agents, automating research pipelines, and reclaiming time for what matters — the patient in front of you.

**Topics covered:**
- Personal AI Infrastructure (PAI) architecture
- Claude Code + MCP server integration in clinical workflows
- Research automation: literature review, data analysis, manuscript drafting
- Live demo: AI-assisted OPD note generation and research query
